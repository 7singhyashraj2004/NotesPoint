# NotesPoint

NotesPoint is a full-stack web application designed to help students access and share study materials easily. Users can upload notes, browse notes by subject, search for relevant content, and download resources shared by the community.

## Features
- User Authentication (Register/Login)
- Upload and manage notes
- Browse notes by category/subject
- Search functionality
- Download study materials
- Responsive UI
- Secure backend APIs

## Tech Stack
- React.js
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- JWT Authentication
