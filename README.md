# NotesPoint

NotesPoint is a MERN-stack based academic notes sharing platform that enables students to upload, manage, discover, and download study materials in a centralized environment. The platform aims to simplify access to educational resources by providing an intuitive interface for sharing notes across different subjects and courses.

The application supports secure user authentication, efficient note management, advanced search capabilities, and organized categorization of study materials. Built with modern web technologies, NotesPoint delivers a responsive and scalable user experience for both contributors and learners.

## Key Features

- Secure user registration and authentication using JWT
- Upload and manage academic notes
- Browse notes by subject and category
- Search notes using keywords
- Download shared study materials
- Responsive design for desktop and mobile devices
- RESTful API architecture
- MongoDB-based data storage
- Clean and user-friendly interface

## Project Motivation

Students often struggle to find quality study materials scattered across multiple platforms. NotesPoint addresses this problem by providing a centralized repository where students can contribute and access educational content efficiently, fostering collaborative learning within academic communities.
